# Interpreter Marketplace

A full-stack web application that connects clients with Dari and Pashto interpreters.

## Features

- Interpreter Registration
- Secure Login with bcrypt
- Dynamic Interpreter Profiles
- Messaging System
- Interpreter Dashboard
- Accept/Reject Requests
